# Ongebaibaenchi Kalenjin (Let's enjoy Kalenjin)
This repo contains a collection of tools and resources for learning (and potentially teaching) Kalenjin, focusing in particular on the Kipsigis dialect.

## Contents

- [Kalenjin Textbook](./textbook/Lesson%2001.md) - A series of dialogs (with audio and some vocabulary and grammar
 notes). The audio can be found [here](https://github.com/chickendude/Ongebaibaenchi-Kalenjin/tree/main/src/textbook/audio).
